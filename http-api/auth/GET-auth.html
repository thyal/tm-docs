GET /auth

https://http.cloud.tiny-mesh.com/v2/auth
200 OK – Responds with information about the current authentication
401 Unauthorized – A Error object due to no authentication provided
403 Forbidden – Fails with a Error object due to failed authentication, or using a non-user token

Description

Retrieves information about the current authentication. Useful to validate that authentication works as expected.
